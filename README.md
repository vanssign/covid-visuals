A [React.js](https://reactjs.org/) based Progressive Web App with redux architecture that fetches covid news and covid cases data from APIs and display it in form of news templates and graphs.

## Frameworks, Libraries and Packages used 

- [React.js](https://reactjs.org/)
- [Bootstrap](https://getbootstrap.com/)
- [Chart.js](https://www.chartjs.org/)
- [React Router](https://reactrouter.com/web/guides/quick-start)
- [Font-Awesome](https://fontawesome.com/)
- [Redux](https://redux.js.org/)

## APIs used
- [api.rootnet.in/](https://api.rootnet.in/covid19-in/stats/)
- [Newscatcher](https://newscatcher.p.rapidapi.com/v1/search_free)

## Deployed on gh-pages
